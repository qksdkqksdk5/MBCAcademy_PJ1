머신러닝 분석훈련 – 핵심 요약

① 군집 분석 (K-Means)

왜 썼나?
자치구를 비슷한 특성끼리 묶어서
지역 유형을 파악하기 위해

사용 변수
평균 생활인구, 평균 자동차 등록 대수

이 분석이 알려주는 것
“서울 자치구는 몇 가지 유형으로 나뉜다”

결과 해석
인구 대비 자동차 보유 특성이 다른
여러 군집이 형성됨

② 선형 회귀 분석

왜 썼나?
인구 변화가 자동차 등록 증감량에
어떤 영향을 주는지 보기 위해

독립 변수 (X)
population_diff (인구 변화량)

종속 변수 (y)
car_diff (자동차 등록 증감량)

이 분석이 알려주는 것
인구가 얼마나 변하면
자동차가 얼마나 변하는지

결과 해석
인구 변화와 자동차 등록 증감 사이에
선형 관계가 존재함

③ 로지스틱 회귀 분석

왜 썼나?
자동차 등록이 증가했는지 여부를
예측하기 위해

타깃 변수
car_diff > 0 → 1
car_diff ≤ 0 → 0

이 분석이 알려주는 것
인구 변화에 따른
자동차 등록 증가 확률

결과 해석
인구가 증가할수록
자동차 등록 증가 확률도 높아지는 경향
